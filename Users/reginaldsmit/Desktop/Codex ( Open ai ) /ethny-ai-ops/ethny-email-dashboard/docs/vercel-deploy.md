# Vercel deployment

1. Push this repository to GitHub.
2. Import it in Vercel and select the main branch.
3. Add the following environment variables in Project Settings > Environment Variables:
   - RESEND_API_KEY
   - RESEND_FROM_EMAIL
   - RESEND_TEST_EMAIL
4. Deploy.

The app is designed to run as a standard Next.js App Router project on Vercel without any special build configuration.
